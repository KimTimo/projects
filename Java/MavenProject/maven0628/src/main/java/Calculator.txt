 중위 표현식 : 연산식에서 연산자를 가운데 둔다. ----> 사람  	예) 3 + 5
 전위 표현식 : 연산식에서 연산자를 앞에 둔다. ---> 컴퓨터		예) + 3 5
 후위 표현식 : 연산식에서 연산자를 뒤에 둔다. ---> 컴퓨터		예) 3 5 +
 
 
 중위 표현식 표현 : 사람
  3 + 5 * 2 = 3 + (5*2) = 3 + 10 = 13
 
 전위 표현식 표현 : 컴퓨터 
 3 + 5 * 2 = (3 + (2 * 5)) = (3 + * 2 5) = + 3 * 2 5 
 
 후위 표현식 표현 : 컴퓨터 
 3 + 5 * 2 = (3 + (2 * 5)) = (3 + 2 5 * ) = 3 2 5 * +
